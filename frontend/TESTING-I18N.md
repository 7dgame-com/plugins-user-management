# 多语言功能测试指南

## 快速测试

### 1. 独立运行测试

启动开发服务器并在 URL 中添加语言参数：

```bash
npm run dev
```

然后在浏览器中访问：
- 简体中文: http://localhost:5173/?lang=zh-CN
- 繁体中文: http://localhost:5173/?lang=zh-TW
- 英语: http://localhost:5173/?lang=en-US
- 日语: http://localhost:5173/?lang=ja-JP
- 泰语: http://localhost:5173/?lang=th-TH

### 2. 在主前端中测试

1. 启动主前端: `cd web && pnpm dev`
2. 登录主系统
3. 在主前端切换语言（右上角语言选择器）
4. 打开用户管理插件
5. 验证插件界面语言与主前端一致
